# 📖 Overview

The **AR Mid Air Drones** project is an interactive Augmented Reality application developed using **Unity 3D** and **Vuforia Engine**. The application enables users to visualize virtual drones hovering and moving dynamically in real-world space through mobile AR technology.

The project demonstrates:
- Real-time Augmented Reality visualization
- Mid-air 3D object rendering
- Interactive drone simulation
- Real-world environment integration
- Immersive AR experiences

The drones appear naturally in the user’s surroundings and can perform animations, movements, and interactive actions in real time.

---

# 📸 How It Works

1. The mobile device camera scans the surrounding environment.
2. The AR engine initializes tracking and scene rendering.
3. Virtual drones are placed dynamically into the environment.
4. The drones hover, animate, and interact in mid-air.
5. Users can move around and observe the drones from different angles in real time.

# 🚀 Installation

## Clone the Repository

```bash
git clone https://github.com/Alok5445Sagar/AR-Mid-Air-Drones.git
```

## Open in Unity

1. Open Unity Hub
2. Add the project folder
3. Open using a compatible Unity version

## Configure Vuforia

- Enable Vuforia Engine
- Configure AR camera settings
- Select Android as build platform

## Run the Application

- Connect Android device
- Build APK
- Launch the AR application
